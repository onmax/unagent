# QStash

> Full CRUD cron scheduling via QStash REST API. The only provider supporting pause/resume.

## Setup

```ts
import { createCron } from 'unagent/cron'

const cron = await createCron({
  provider: {
    name: 'qstash',
    token: process.env.QSTASH_TOKEN!,
  },
})
```

## Create Schedule

```ts
const schedule = await cron.create({
  cron: '*/5 * * * *',
  destination: 'https://example.com/api/task',
  method: 'POST',
  body: { key: 'value' },
  headers: { 'x-custom': 'header' },
  retries: 3,
  timeout: '30s',
})
```

Use `scheduleId` to upsert (update existing schedule):

```ts
const schedule = await cron.create({
  cron: '0 * * * *',
  destination: 'https://example.com/api/task',
  scheduleId: 'my-schedule',
})
```

## List Schedules

```ts
const schedules = await cron.list()
```

## Get Schedule

```ts
const schedule = await cron.get('sched_123')
```

## Delete Schedule

```ts
await cron.delete('sched_123')
```

## Pause / Resume

```ts
await cron.pause('sched_123')
await cron.resume('sched_123')
```

## Capabilities

All operations supported:

```ts
cron.supports
// { create: true, list: true, get: true, delete: true, pause: true, resume: true }
```
