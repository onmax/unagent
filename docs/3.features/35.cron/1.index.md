# Cron

> Schedule recurring tasks with a unified API across multiple providers.

## Usage

```ts
import { createCron } from 'unagent/cron'
```

## Quick Start

```ts
const cron = await createCron({
  provider: { name: 'qstash', token: process.env.QSTASH_TOKEN! },
})

// Create a schedule
const schedule = await cron.create({
  cron: '*/5 * * * *',
  destination: 'https://example.com/api/task',
})

// List all schedules
const schedules = await cron.list()

// Delete a schedule
await cron.delete(schedule.id)
```

## Build Helpers

Generate platform-specific cron configuration at build time:

```ts
import { defineCronSchedules, toCloudflareCrons, toVercelCrons } from 'unagent/cron'

const schedules = defineCronSchedules([
  { cron: '*/5 * * * *', path: '/api/cleanup' },
  { cron: '0 0 * * *', path: '/api/daily-report' },
])

// For wrangler.toml [triggers].crons
const cfCrons = toCloudflareCrons(schedules)

// For vercel.json crons
const vercelCrons = toVercelCrons(schedules)
```
