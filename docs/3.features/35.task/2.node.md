# Node / Bun / Deno

> In-process cron scheduling powered by [croner](https://github.com/hexagon/croner).

## Usage

```ts
import { createTaskRunner } from 'unagent/task'

const cleanup = {
  meta: { description: 'Clean up stale records' },
  async run() {
    return { result: { deleted: 42 } }
  },
}

const runner = createTaskRunner({
  tasks: { 'db:cleanup': cleanup },
  scheduledTasks: { '0 */6 * * *': 'db:cleanup' },
})

const stop = runner.startScheduler()

// Stop when shutting down
process.on('SIGTERM', stop)
```

The scheduler runs in the same process. Each cron tick calls `runTask()`, which benefits from concurrent deduplication.
