# Vector

> Unified vector database adapters with a text-first API and optional embeddings.

## Usage

```ts
import { createVector } from 'unagent/vector'
```

## Quick Start

```ts
const vector = await createVector({
  provider: {
    name: 'upstash',
    url: process.env.UPSTASH_VECTOR_URL!,
    token: process.env.UPSTASH_VECTOR_TOKEN!,
    namespace: 'chunks',
  },
})

await vector.index([{ id: '1', content: 'hello world' }])
const results = await vector.search('hello')
```

## API

```ts
interface VectorClient {
  readonly provider: string
  readonly supports: { deleteById: boolean, returnContent: boolean }

  index: (items: { id: string, content: string, metadata?: Record<string, unknown> }[]) => Promise<void>
  search: (query: string, opts?: Record<string, unknown>) => Promise<unknown[]>
  delete?: (ids: string[]) => Promise<void>
}
```
