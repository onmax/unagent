# Deno

Deno-specific sandbox features are available through the `sandbox.deno` namespace.

## Setup

```ts
import { createSandbox } from 'unagent/sandbox'

const sandbox = await createSandbox({
  provider: { name: 'deno', timeout: '5m' },
})
```

::note
Deno Deploy sandboxes require a valid `DENO_DEPLOY_TOKEN` in the environment.
::

## API

### DenoNamespace API

#### runtime

Access the Deno runtime helper:

```ts
const runtime = sandbox.deno.runtime
```

#### env

Read and write environment variables inside the sandbox:

```ts
await sandbox.deno.env.set('HELLO', 'world')
const value = await sandbox.deno.env.get('HELLO')
```

#### fs

Access the sandbox filesystem directly:

```ts
await sandbox.deno.fs.writeTextFile('/tmp/demo.txt', 'hello')
const text = await sandbox.deno.fs.readTextFile('/tmp/demo.txt')
```

#### exposeHttp

Expose a sandbox port through Deno Deploy:

```ts
const url = await sandbox.deno.exposeHttp({ port: 8080 })
```

#### exposeSsh

Get SSH access for debugging:

```ts
const ssh = await sandbox.deno.exposeSsh()
// { hostname, username }
```

#### exposeVscode

Open a remote VS Code session:

```ts
const vscode = await sandbox.deno.exposeVscode('/app')
```

#### extendTimeout

Extend the sandbox timeout:

```ts
await sandbox.deno.extendTimeout('10m')
```

#### fetch

Perform HTTP requests from the sandbox network:

```ts
const res = await sandbox.deno.fetch('https://example.com')
```

#### spawn

Start a long-running process:

```ts
const proc = await sandbox.deno.spawn('deno', { args: ['--version'] })
```

#### close and kill

Close or terminate the sandbox:

```ts
await sandbox.deno.close()
await sandbox.deno.kill()
```

## Example

```ts
import { createSandbox } from 'unagent/sandbox'

const sandbox = await createSandbox({
  provider: { name: 'deno', timeout: '5m' },
})

try {
  const output = await sandbox.exec('echo', ['hello from deno'])
  const url = await sandbox.deno.exposeHttp({ port: 8080 })
  return { output, url }
}
finally {
  await sandbox.stop()
}
```
