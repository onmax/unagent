# Workflow

> Create durable workflows with a unified API across multiple providers.

## Usage

```ts
import { createWorkflow } from 'unagent/workflow'
```

## Quick Start

```ts
const workflow = await createWorkflow({
  provider: { name: 'vercel', workflow: userSignupWorkflow },
})

const run = await workflow.start(['user@example.com'])
const status = await run.status()
await run.stop()
```

## Providers

- [Vercel](/features/workflow/vercel)
- [Cloudflare](/features/workflow/cloudflare)

## API

### Unified API

```ts
interface WorkflowClient {
  readonly provider: 'vercel' | 'cloudflare' | 'openworkflow'
  readonly supports: { startBatch: boolean }

  start: (payload?: unknown, options?: Record<string, unknown>) => Promise<WorkflowRun>
  get: (id: string) => Promise<WorkflowRun | null>
  startBatch?: (items: unknown[]) => Promise<WorkflowRun[]>
}
```
