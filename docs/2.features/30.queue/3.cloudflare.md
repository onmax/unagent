# Cloudflare

> Cloudflare-specific queue features via the `queue.cloudflare` namespace.

## Usage

```ts
import { createQueue } from 'unagent/queue'

const queue = await createQueue({
  provider: { name: 'cloudflare', binding: env.MY_QUEUE },
})

await queue.send({ hello: 'world' }, { contentType: 'json', delaySeconds: 10 })
await queue.sendBatch?.([{ body: { ok: true }, contentType: 'json' }], { delaySeconds: 10 })
```

## Consumer

```ts
import { createCloudflareQueueBatchHandler } from 'unagent/queue'

export default {
  queue: createCloudflareQueueBatchHandler({
    async onMessage(message) {
      console.log(message.body)
    },
  }),
}
```

## API

### CloudflareNamespace API

```ts
const cf = queue.cloudflare
cf.binding // Queue binding instance
```
