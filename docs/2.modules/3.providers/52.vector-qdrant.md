# vector-qdrant

> Qdrant provider for `unagent/vector`.

## Usage

### Setup
```ts
import { createVector } from 'unagent/vector'
import { ollama } from 'unagent/vector/embeddings/ollama'

const vector = await createVector({
  provider: {
    name: 'qdrant',
    url: 'http://localhost:6333',
    collection: 'vectors',
    embeddings: ollama({ model: 'nomic-embed-text' }),
  },
})
```

## API

### Requirements
- A running Qdrant instance
- `QDRANT_URL` (optional if using the Docker compose)

### Namespace
```ts
const client = vector.qdrant.client
```

### E2E testing
Use the docker compose (includes Qdrant) and run:

```bash
VECTOR_E2E_DOCKER=1 pnpm playground:vector:e2e
```

Or run against a remote Qdrant:

```bash
QDRANT_URL=https://your-qdrant.example.com pnpm playground:vector:e2e
```
