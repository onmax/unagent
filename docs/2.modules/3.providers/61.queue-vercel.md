# queue-vercel

> Vercel-specific queue features via the `queue.vercel` namespace.

## Usage

```ts
import { createQueue } from 'unagent/queue'

const queue = await createQueue({
  provider: { name: 'vercel', topic: 'my-topic' },
})

const { messageId } = await queue.send({ hello: 'world' }, {
  idempotencyKey: 'unique-key',
  retentionSeconds: 3600,
  delaySeconds: 10,
})
```

## API

### VercelNamespace API

```ts
const vercel = queue.vercel
vercel.topic // configured topic
vercel.native // access to @vercel/queue
```

### Limitations
- `sendBatch()` is not supported on Vercel (use `send()`).
