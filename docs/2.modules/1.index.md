# Modules

The `unagent` library organizes its functionality into focused modules. Each module exports separately to enable tree-shaking and reduce bundle size.

## Available Modules

| Module | Import | Description |
|--------|--------|-------------|
| [context](/modules/context) | `unagent/context` | Token estimation, truncation, context building |
| [env](/modules/env) | `unagent/env` | Agent detection, config paths, XDG |
| [exec](/modules/exec) | `unagent/exec` | Safe command execution with timeout |
| [fs](/modules/fs) | `unagent/fs` | Virtual filesystem (in-memory, overlay, real) |
| [git](/modules/git) | `unagent/git` | Clone repos, git operations |
| [hooks](/modules/hooks) | `unagent/hooks` | Agent lifecycle hooks |
| [link](/modules/link) | `unagent/link` | Copy directories, symlinks |
| [lock](/modules/lock) | `unagent/lock` | Skill lockfile management |
| [registry](/modules/registry) | `unagent/registry` | Provider registry with priority matching |
| [sandbox](/modules/sandbox) | `unagent/sandbox` | Sandboxed execution environments |
| [skill](/modules/skill) | `unagent/skill` | Skill discovery, parsing, install/uninstall |
| [source](/modules/source) | `unagent/source` | URL and source string parsing |
| [stop](/modules/stop) | `unagent/stop` | Stop conditions for agent loops |
| [stream](/modules/stream) | `unagent/stream` | Stream handling, SSE formatting/parsing |
| [tool](/modules/tool) | `unagent/tool` | Define tools, convert to JSON Schema |
| [usage](/modules/usage) | `unagent/usage` | Token usage tracking, cost calculation |
| [utils](/modules/utils) | `unagent/utils` | ANSI colors, formatting, paths |

## Full Import

The main entry point exports all public APIs from every module.

```ts
import { detectCurrentAgent, discoverSkills, parseSource } from 'unagent'
```

## Selective Import

Individual module imports reduce bundle size by including only the code you need.

```ts
import { detectCurrentAgent } from 'unagent/env'
import { discoverSkills } from 'unagent/skill'
import { parseSource } from 'unagent/source'
```
