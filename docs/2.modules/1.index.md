## Full Import

The main entry point exports all public APIs from every module.

```ts
import { detectCurrentAgent, discoverSkills, parseSource } from 'unagent'
```

## Selective Import

Individual module imports reduce bundle size by including only the code you need.

```ts
import { detectCurrentAgent } from 'unagent/env'
import { discoverSkills } from 'unagent/skill'
import { parseSource } from 'unagent/source'
```
