# Modules

`unagent` is organized into focused modules, each exported separately for tree-shaking.

## Available Modules

| Module | Import | Description |
|--------|--------|-------------|
| [env](/modules/env) | `unagents/env` | Agent detection, config paths, XDG |
| [skill](/modules/skill) | `unagents/skill` | Skill discovery and markdown parsing |
| [source](/modules/source) | `unagents/source` | URL and source string parsing |
| [git](/modules/git) | `unagents/git` | Clone repos, git operations |
| [link](/modules/link) | `unagents/link` | Copy directories, symlinks |
| [lock](/modules/lock) | `unagents/lock` | Skill lockfile management |
| [utils](/modules/utils) | `unagents/utils` | ANSI colors, formatting, paths |

## Full Import

Import everything from the main entry:

```ts
import { detectCurrentAgent, parseSource, discoverSkills } from 'unagents'
```

## Selective Import

Import only what you need for smaller bundles:

```ts
import { detectCurrentAgent } from 'unagents/env'
import { parseSource } from 'unagents/source'
import { discoverSkills } from 'unagents/skill'
```
