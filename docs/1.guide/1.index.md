# Getting Started

`unagent` provides unified primitives for building AI coding agent tooling. Detect which agent is running, discover skills, parse sources, and manage installations.

## Features

- [**Agent Detection**](/modules/env) - Detect 40+ AI coding agents by env vars or config
- [**Skill Discovery**](/modules/skill) - Parse and discover markdown-based skill files
- [**Source Parsing**](/modules/source) - Parse GitHub/GitLab URLs, owner/repo shortcuts, local paths
- [**Git Operations**](/modules/git) - Clone repos, manage temp dirs, check status
- [**File Linking**](/modules/link) - Copy directories, create symlinks safely
- [**Lockfile Management**](/modules/lock) - Track installed skills with hashes

## Quick Example

```ts [detect.ts]
import { detectCurrentAgent, detectInstalledAgents } from 'unagent/env'

// Detect which agent is currently running
const agent = detectCurrentAgent()
if (agent) {
  console.log(`Running in ${agent.config.name}`)
}

// Find all installed agents
const installed = detectInstalledAgents()
console.log(`Found ${installed.length} agents`)
```

```ts [skills.ts]
import { discoverSkills, parseSkillMd } from 'unagent/skill'

// Discover skills in a directory
const skills = discoverSkills('~/.claude/.agents/skills', { recursive: true })
// Legacy 'skills' directory remains supported for backward compatibility.

// Parse a skill file
const parsed = parseSkillMd(`---
name: My Skill
globs: ["*.ts"]
---
Instructions for the agent...
`)
```

```ts [source.ts]
import { parseSource } from 'unagent/source'

// Parse various source formats
parseSource('unjs/unagent') // → github owner/repo
parseSource('github:user/repo#main') // → github with ref
parseSource('./local/path') // → local path
```
