# Installation

> Install `unagent` and import only the modules you need.

## Package Manager

:pm-install{name="unagent"}

## Minimal Setup

```ts
import { detectCurrentAgent } from 'unagent/env'

const agent = detectCurrentAgent()
console.log(agent?.config.name)
```
